Troll TV is a retro procedurally generated rogue-lite turn based rpg with persistent leveling. It draws inspiration from games like Desktop Dungeons, Lost Labyrinth, and of course Rogue.

Troll TV uses a soft version of perma-death: Each play will start you without any items or experience points earned in the previous runs. The only exception to this is your total winnings which can be used to buy special abilities at the beginning of each run. You can freely swap these each time so you'll never be locked into a single build.

Your total winnings will automatically be saved each time you retire or die, so as long as you play on the same computer you'll only have to start with absolutely nothing once.
